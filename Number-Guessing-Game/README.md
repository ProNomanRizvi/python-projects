# Number Guessing Game

A simple interactive Number Guessing Game implemented in a Jupyter Notebook.

## Description
The computer randomly selects a number between 1 and 100. Your goal is to guess the correct number within 5 attempts. After each guess, the game will tell you if your guess was too high or too low.

## How to Play
1. Run the notebook cells in order.
2. When prompted, enter a number between 1 and 100.
3. Use the feedback ("Too low" or "Too high") to adjust your next guess.
4. Win by guessing the correct number before your attempts run out!

## Requirements
- Python 3
- Jupyter Notebook
